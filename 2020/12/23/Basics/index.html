Russel Antinomy. The predicate $P(x):x\notin x$ does not define a set $A=\{x:P(x)\}$.

Proof:

  1. Assume $A\in A$. Then, according to $P(x)$, $A\notin A$.

  2. Assume $A\notin A$. Then, according to $\neg P(x)$, $A\in A$.

Sequences.

General Results:

  1. A convergent sequence is bounded.
  2. A convergent sequence has precisely one limit.
  3. If $\lim a_n=a,\lim b_n=b$, then
    1. $\lim(a_n+b_n)=a+b$
    2. $\lim(a_n\cdot b_n)=a\cdot b$
    3. $\lim(a_n/b_n)=a/b$, if $b\ne0$

Squeeze Theorem. Let $(a_n),(b_n),(c_n)$ be real sequences with $a_n<c_n<b_n$ for sufficiently large $n$. Suppose that $\lim a_n=\lim b_n=a$. Then, $(c_n)$ converges and $\lim c_n=a$.

A Useful Lemma from Continuity. Let $\Omega\subset\mathbb{R}$ be some set, $f:\Omega\rightarrow\mathbb{R}$ a function that is continuous at some point $x_0\in\Omega$ and assume that $f(x_0)>0$. Then there exists a $\delta>0$ such that $f(x)>0$ for all $x\in(x_0-\delta,x+\delta)\cap\Omega$.

Zero Value Theorem. Let $a<b$ and $f:[a,b]\rightarrow\mathbb{R}$ be a continuous function with $f(a)<0<f(b)$. There exists some $x\in[a,b]$ such that $f(x)=0$.

Bolzano Intermediate Value Theorem. Let $a<b$ and $f:[a,b]\rightarrow\mathbb{R}$ be a continuous function. Then for $y\in[\min\{f(a),f(b)\},\max\{f(a),f(b)\}]$, there exists some $x\in[a,b]$ such that $f(x)=y$.

Boundedness of Continuous Function. Let $a<b$ and $f:[a,b]\rightarrow\mathbb{R}$ be a continuous function. Then there exists a $y\in[a,b]$ such that $f(x)<f(y)$ for all $x\in[a,b]$.
