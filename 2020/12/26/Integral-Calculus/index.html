Integral Calculus

Posted on 2020-12-26 Edited on 2021-01-02 In math tool

A collection of theorems in Integral Calculus in one variable based on the third part of the course VV186 in UM-SJTU JI.

Integrations.

The Fundamental Theorem of Calculus. Let \(f:[a,b]\rightarrow\mathbb{C}\) be continuous and set \[ F:[a,b]\rightarrow\mathbb{C},\ \ \ \ \ F(x):=\int\limits_a^xf \] Then \(F\) is differentiable on \((a,b)\) and \[ F'(x)=f(x),\ \ \ \ x\in(a,b) \]

Substitution Rule. Let \(f\in\text{Reg}([\alpha,\beta])\) and \(g:[a,b]\rightarrow[\alpha,\beta]\) continuously differentiable. Then \[ \int\limits_a^b(f\circ g)(x)g'(x)dx=\int\limits_{g(a)}^{g(b)}f(y)dy \]

Integration by Parts. Let \(f,g\in C^1([a,b],\mathbb{C})\). Then \[ \int\limits_a^bf'(x)g(x)dx=f(x)g(x)|^b_a-\int\limits^b_af(x)g'(x)dx \]

Cauchy Property. Let \(I\subset\mathbb{R}\) be an interval, \(a\in\bar{I}\subset(\mathbb{R}\ \cup \{-\infty,\infty\})\) and \(F:I\rightarrow\mathbb{C}\). Then the following statements are equivalent:

  1. The limits \(\lim\limits_{x\rightarrow a}F(x)\) exists.

  2. The function satisfies Cauchy property: \[ \underset{\varepsilon>0}{\forall}\ \underset{\delta>0}{\exists}\ \underset{x,y \in I}{\forall}:|x-a|<\delta \wedge|y-a|<\delta \Rightarrow|F(x)-F(y)|<\varepsilon \] when \(a\in\mathbb{R}\) or \[ \underset{\varepsilon>0}{\forall}\ \underset{R>0 }{\exists}\ \underset{x,y \in I}{\forall}: \pm x, \pm y>R \Rightarrow|F(x)-F(y)|<\varepsilon \] when \(a=\pm\infty\)

Cauchy Criterion. Let \(a\in\mathbb{R}\) and \(f:[a,\infty)\rightarrow\mathbb{R}\) be integrable on every interval \([a,x]\), \(x\in\mathbb{R}\). The improper integral \[ \int\limits_a^\infty f(x)dx \] converges if and only if \[ \underset{\varepsilon>0}{\forall}\ \underset{R>0}{\exists}\ \underset{x,y>R}{\forall}\ \left|\int_{x}^{y} f(t) d t\right|<\varepsilon \]

The Dirichlet Integral. \[ \int\limits_0^\infty \frac{\sin x}{x}dx \] is convergent.

Comparison Test. Let \(I\subset\mathbb{R}\) and \(f:I\rightarrow\mathbb{C}\), \(g:I\rightarrow[0,\infty)\). Suppose that \(|f(t)|\leq g(t)\) for all \(t\in I\) and \(\int_Ig(t)dt\) converges. Then \(\int_If(t)dt\) also converges.

Euler Gamma Function. is defined through \[ \Gamma:\mathbb{R}_+\rightarrow\mathbb{R},\ \ \ \ \ \Gamma(t):=\int\limits_0^\infty z^{t-1}e^{-z}dz,\ \ \ t>0 \] The gamma function plays an important role in many parts of mathematics, physics, and engineering.

Here we list some properties of gamma function.

  1. The improper integral which defines the gamma function converges absolutely for all \(t>0\)

  2. \(\Gamma(t+1)=t\Gamma(t)\) for all \(t>0\)

  3. Using property 2 we can further extend gamma function to \(t<0\) by \[ \Gamma(t):=\frac{\Gamma(t+1)}{t},\ \ \ t\in(-n-1,-n),\ n\in\mathbb{N} \]

Integral Test. Let \(m\in\mathbb{N}\) and \(f:[m,\infty)\rightarrow[0,\infty)\) be a decreasing function such that \(\int\limits_m^xf(x)\) exists for all \(x>m\). Then \[ \int\limits_m^\infty f(x)dx<\infty\ \ \ \ \Leftrightarrow\ \ \ \ \sum\limits_{n=m}^\infty f(n)<\infty \]

Taylor's Theorem. Let \(I\in\mathbb{R}\) be an open interval and \(f\in C^k(I)\). Let \(x\in I\) and \(y\in\mathbb{R}\) such that \(x+y\in I\). Then for all \(p\le k\), \[ f(x+y)=f(x)+\frac{1}{1!}f'(x)y+\frac{1}{2!}f''(x)y^2+\cdots+\frac{1}{(p-1)!}f^{(p-1)}(x)y^{p-1}+R_p \] with the reminder term (Lagrange form) \[ R_p:=\int\limits_0^1\frac{(1-t)^{p-1}}{(p-1)!}f^{(p)}(x+ty)y^pdt \]

Abel's Limit Theorem. Let \(f:[0,1)\rightarrow\mathbb{R}\) be defined by \[ f(x)=\sum\limits_{k=0}^\infty a_kx^k \] where \(\sum a_k\) converges. Then \(\sum a_kx^k\) converges uniformly on \([0,1]\), so \(f\in C([0,1])\) and \[ \sum\limits_{k=0}^\infty a_k=\lim_{x\nearrow1}\sum_{k=0}^{\infty} a_k x^k \] Here define the Abel Sum \[ \mathrm{A}-\sum_{n=0}^{\infty} a_{n}:=\lim_{x\nearrow1}\sum_{k=0}^{\infty} a_k x^k \] With such summability, some interesting fact can be found: \[ \mathrm{A}-\sum_{n=0}^{\infty} (-1)^n = \frac{1}{2}\\ \mathrm{A}-\sum_{n=0}^{\infty} n(-1)^{n+1} = \frac{1}{4} \]

Riemann Zeta Function. is defined as \[ \zeta(s)=\sum\limits_{n=1}^\infty \frac{1}{n^s} \] This function will have more applications in more advanced topics. We omit them here.

The graph of zeta function is shown below.

Stirling's Formula. As \(n\rightarrow\infty\) \[ n ! \sim \sqrt{2 \pi n}\left(\frac{n}{e}\right)^{n} \] This formula is a good elementary estimate to factorial.
